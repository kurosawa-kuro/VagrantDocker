# Vagrant

## Vagrantで仮想環境作成

```
vagrant up
```

bootstrapが反応してprovisionされ、ミドルウェア・ライブラリがインストールされます。

## Vagrantで作成された仮想環境の停止、削除

- 停止

```
vagrant halt
```

- 削除

```
vagrant destroy
```
